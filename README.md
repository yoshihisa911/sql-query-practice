# sql-practice

This repository contains SQL practice projects using orders, users, and payments tables.

This query calculates total spending by customer, finds customers whose total spending is above the average, and keeps only customers with at least one order over 200,000.

Topics:
- JOIN
- GROUP BY
- HAVING
- CASE WHEN
- Correlated Subqueries

The goal is to practice SQL queries and improve data analysis skills.
